用javascript html css 寫在同一的檔案，使用emji當作圖示
我們先從框架開始 遊戲使用整個畫面

開始狀態：
	顯示一個按鈕開始遊戲 點擊按鈕進入遊戲狀態
遊戲狀態：
	水果從下方丟到到畫面上，水果因為地心引力的加速速落下
	計算丟出畫面的速度使得水果高度超過畫面1/2～頂部高度才落下
	水果需要緩緩地旋轉，需要有左右移動的速度，但最終落下的Ｘ不能超出螢幕外
	水果的大小為120px
	左上角顯示分數，右上角顯示三個愛心
	水果落下低於畫面底部失去一個愛心
	三個愛心都失去進入遊戲結束狀態
	
	滑鼠指標移動的時候要有拖尾的效果，
	透過滑鼠的位置和上一次的滑鼠位置當作刀的位置，
	實現精確的碰撞檢測演算法，考慮水果的實際大小和位置。
	
	當刀子切中水果時，在切到的位置顯示一條長直線，延伸長度為100至200，
	代表刀切的方向。這條直線只顯示一瞬間，隨即消失。
	被切中的水果轉變成多個圓型的水果汁,水果物件立刻刪除不要透明特效．
	
	除了要有各種水果還要有炸彈，
	殺中炸彈會螢幕震動扣一顆心，炸彈掉下不會扣分

戲結束狀態： 
	顯示GAME OVER與分數，等待三秒回到開始狀態
